# Base64 Codec — Frontend

A simple static webpage that provides a UI for a Base64 encode/decode REST API.

## Features

- **Encode / Decode text** — paste text and get the result inline
- **Encode / Decode files (buffered)** — upload files up to 2 MB; result is downloaded automatically
- **Encode / Decode files (streamed)** — same flow, supports files up to 50 MB
- Copy buttons on every input/output field
- Error dialog for API error responses

## API

You can check out the API project at the url: [Base64-codec](https://github.com/gabrielmlborges/Base64-codec).

The API is running on: https://base64codedapi.onrender.com
